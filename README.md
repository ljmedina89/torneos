# Torneos

Plataforma deportiva local para campeonatos de fútbol y otros deportes en Ecuador.

## Objetivo del MVP

Crear una web moderna, rápida y compartible para mostrar:

- Resultados por jornada
- Tabla de posiciones
- Próximos partidos
- Equipos participantes
- Tabla de goleadores
- Postales visuales para redes sociales
- Panel administrativo básico

## Visión

Convertir campeonatos locales, barriales y parroquiales en experiencias digitales profesionales, con resultados, estadísticas, contenido visual, transmisión y patrocinadores.

## Stack inicial

- HTML
- CSS
- JavaScript vanilla
- Google Sheets como base de datos
- Google Apps Script como API
- GitHub Pages como hosting

## Prioridad inmediata

Primero construir una versión funcional, visual y limpia. No sobrecomplicar.

El primer lanzamiento debe permitir mostrar un campeonato real aunque los datos estén cargados manualmente o desde una hoja de Google Sheets.
