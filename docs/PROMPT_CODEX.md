# PROMPT MAESTRO PARA CODEX

Construye una plataforma web moderna llamada "Torneos" enfocada en campeonatos deportivos locales de Ecuador.

## Objetivo

Crear un MVP funcional y visualmente atractivo para:

- Mostrar resultados deportivos
- Mostrar tabla de posiciones
- Mostrar goleadores
- Mostrar próximos partidos
- Administrar jornadas y resultados
- Generar contenido visual para redes sociales

## Diseño

Inspiración visual:

- ESPN
- Sofascore
- Flashscore
- LigaPro

Estilo:

- Moderno
- Oscuro
- Deportivo
- Responsive
- Optimizado para móviles

## Tecnologías

Frontend:
- HTML
- CSS
- JavaScript vanilla

Backend:
- Google Sheets
- Apps Script API

Hosting:
- GitHub Pages

## Módulos MVP

### Público

1. Home
2. Tabla de posiciones
3. Resultados por jornada
4. Próximos partidos
5. Equipos
6. Goleadores

### Admin

1. Registrar resultados
2. Crear jornadas
3. Registrar equipos
4. Registrar goleadores

## Objetivo técnico

Construir primero una versión funcional y elegante.

Evitar:
- frameworks innecesarios
- backend complejo
- autenticación avanzada
- sobreingeniería

Priorizar:
- velocidad
- estética
- simplicidad
- facilidad de mantenimiento

## Objetivo futuro

Escalar a:
- múltiples campeonatos
- streaming deportivo
- patrocinadores
- perfiles de jugadores
- estadísticas avanzadas
- sistema SaaS multiusuario
